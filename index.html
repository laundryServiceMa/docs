Stores

Stores

Those are the stores (Locals) that has the tablets which runs the machiens, each one has a Name, Current income

Edit store

You can edit the store by clicking on the store, and then you can edit the store and it will be auto saved.

Machines

Machines

Those are the machines that are in the stores, each one has a Name, Status, Store, Price, active, the status is the status of the machine if its running or not, the active is if the machine is active or not (showing it to the user), the price is the price of the machine, the store is the store that the machine is in.

Edit machine

You can edit the machine by clicking on the machine, and then you can edit the machine and it will be auto saved, and you change its price category from here

Edit tablet

You can edit the tablet by clicking on the tablet, and then you can edit the tablet and it will be auto saved.
